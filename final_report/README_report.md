# Final Report Package

This folder contains the clean report deliverables for submission and GitHub review.

- `Reconstructing_Wonderland_Case_Level_Report.docx`: final case-level Word report.
- `part1_case_level_analysis_draft.md`: expanded Part 1 analysis draft used while developing the report.

The report can be regenerated from `scripts/build_case_level_word_report.py`. Temporary render QA folders and intermediate generated report figures are intentionally ignored by Git.
